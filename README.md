# Actividad-evaluacion
## Retrospectiva del proyecto

Durante este proyecto he desarrollado una página web estilo plataforma de streaming (F5Flix) utilizando HTML y CSS, aplicando también conceptos básicos de JavaScript para mejorar la interactividad.

He creado varias secciones como el catálogo de películas, series, “lo más visto”, así como páginas adicionales como detalle, registro y contacto. También he implementado un diseño responsive para que la web se adapte a distintos dispositivos.

Uno de los aspectos más importantes que he aprendido ha sido el uso de CSS Grid y Flexbox para estructurar correctamente los elementos, así como la organización del código en diferentes carpetas para mantener un proyecto limpio y escalable.

También he trabajado en mejorar la experiencia de usuario mediante efectos hover, menús desplegables y un diseño visual más cercano a plataformas reales como Netflix.

La mayor dificultad ha sido gestionar el diseño responsive y evitar que los elementos se descoloquen en diferentes tamaños de pantalla. Además, ajustar detalles visuales como alineaciones y overlays también ha requerido bastante trabajo.

En general, me ha parecido un proyecto muy interesante porque me ha permitido aplicar de forma práctica muchos conceptos y ver cómo se construye una web más completa y profesional.

## Conclusión
Al hacer las capturas de pantalla, me di cuenta de que el diseño se ve bastante bien en diferentes dispositivos, aunque todavía hay algunos detalles que podrían mejorarse para una experiencia más fluida. En general, estoy satisfecho con el resultado final y creo que he aprendido mucho durante este proceso.
